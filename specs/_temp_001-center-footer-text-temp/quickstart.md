# Quickstart: Center Footer Text

## Overview
This feature implements horizontal centering for specific text elements in the website footer: "Physical AI & Humanoid Robotics", "Master embodied intelligence with hands-on labs", and "Building the Future of Embodied Intelligence".

## Implementation Steps
1. Locate the footer component in the Docusaurus theme
2. Identify the CSS selectors for the target text elements
3. Apply centering styles using CSS
4. Test responsiveness across device sizes

## Files to Modify
- `frontend/src/theme/Footer/Logo/index.js` (or related footer component)
- `frontend/src/css/custom.css` (or new CSS file for footer styles)

## Testing
1. Verify text is centered on desktop
2. Verify text remains centered on mobile and tablet
3. Check that centering works across all website pages
4. Ensure no layout issues on different screen sizes
